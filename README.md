# Videogames-and-Juvenile-Violent-Crime-Panel-Analysis
State-level panel analysis of video game exposure and juvenile violent crime using IPUMS CPS and FBI UCR data (1984–2003).

# Video Game Exposure and Juvenile Violent Crime

## Overview

This repository contains the full replication pipeline for an empirical study
examining the relationship between video game exposure and juvenile violent crime
rates in the United States.

The analysis combines:

- IPUMS CPS microdata (individual-level media exposure proxies)
- FBI Uniform Crime Reports (UCR) arrest panel data
- State-level panel construction (1984–2003)
- Fixed-effects regression framework

All data processing, merging, and estimation are reproducible through a
modular R pipeline via `run_all.R`.

## Repository Structure – Key Modules
### Root-Level Files

`README.md` – Project documentation
Overview, data sources, workflow, and reproducibility instructions.

`run_all.R` – Master execution script
Runs the full empirical pipeline in sequence: data processing, panel construction, and regressions.

`.gitignore` – Git exclusion rules
Prevents raw data, processed outputs, and system files from being committed.

`renv.lock` – Reproducible package snapshot
Locks R package versions to ensure full replication.

## Data Layer
### Raw Data (Not Committed)

Raw data is not committed to this repository due to size constraints and best practices.

`data/raw/ipums/CPSFull.csv`

IPUMS CPS microdata extract used to construct demographic controls and exposure proxies.

`data/raw/ucr/`

FBI Uniform Crime Reports (UCR) arrest data in Stata .dta format (monthly agency-level data).

### Processed Data Outputs

These files are generated by the pipeline.

`data/processed/CPSProcessed.csv`
Cleaned and filtered IPUMS dataset.

`data/processed/ipums_clean.rds`
Serialized R version of cleaned IPUMS data for fast loading.

`data/processed/ucr_clean_1984_2003.rds`
Cleaned UCR dataset restricted to agencies reporting 12 months and analysis years.

`data/processed/merged_panel.rds`
Final state-year panel dataset used for regression analysis.

## Core Processing Modules (src/)
### Configuration & Utilities

`src/config.R` – Centralized configuration
Defines file paths, analysis years, constants, and directory structure.

`src/utils.R` – Helper functions
Reusable data-cleaning, aggregation, and transformation functions.

## Data Pipelines

### IPUMS Processing

`src/ipums_pipeline.R`

Loads raw CPS microdata
Cleans and filters variables
Constructs exposure measures
Saves processed outputs

### UCR Processing

`src/ucr_pipeline.R`

Loads yearly UCR .dta files
Filters agencies reporting full 12 months
Constructs juvenile violent crime measures
Saves cleaned dataset

### Panel Construction

`src/merge_panel.R`

Aggregates IPUMS data to state-year level
Merges with UCR crime measures
Constructs final analysis panel
Outputs merged dataset

### Econometric Analysis

`src/regressions.R`

Estimates fixed-effects panel regressions
Applies clustered standard errors
Runs robustness checks
Outputs regression tables and figures

## Full Script Archive

`full_pipeline/full_pipeline.R`

Original monolithic script containing the complete analysis in one file.
Maintained for archival and reference purposes.

## Typical Workflow
### Data Acquisition

Place raw IPUMS CPS and UCR files in data/raw/.

### Data Processing

Run:
`source("run_all.R")`

This executes:

  - IPUMS data cleaning
  - UCR data cleaning
  - Panel construction
  - Econometric estimation

### Outputs

Processed datasets are saved to `data/processed/`.
Regression outputs are saved to `results/`.
